# 表单自动填充助手 - 隐私权规范

## 单一用途说明

本扩展的唯一用途是：**自动识别网页表单并生成模拟数据填充，方便开发者和测试人员快速测试表单功能**。

本扩展不会：

- 收集、存储或传输用户的个人数据
- 跟踪用户的浏览行为
- 向第三方发送用户数据
- 修改或删除用户数据
- 执行任何与表单自动填充无关的功能

## 权限使用说明

### 1. 主机权限（host_permissions: http://_/_, https://_/_）

**使用理由：**
本扩展需要访问所有网站（http://_/_ 和 https://_/_），以便在任意网页上识别和填充表单元素。由于用户可能在任何网站上使用表单功能，因此需要通用主机权限来支持所有网站。

**具体用途：**

- 检测页面上的表单元素（input、textarea、select 等）
- 读取表单字段的标签（label）、名称（name）、占位符（placeholder）等信息
- 将生成的模拟数据填充到表单字段中

**数据访问：**

- 本扩展仅读取表单字段的结构信息（如字段名称、类型、标签），不读取用户已输入的数据
- 不会访问或读取页面的其他内容
- 不会访问或读取用户的个人信息

### 2. activeTab 权限

**使用理由：**
本扩展需要访问当前活动标签页的内容，以便：

1. 检测页面上是否存在表单元素
2. 提取表单字段的元数据（label、name、placeholder 等）
3. 将生成的模拟数据填充到表单字段中

**具体用途：**

- 仅在用户主动点击"自动填充"按钮时，访问当前标签页的 DOM 结构
- 仅读取表单元素的属性信息，不读取用户已输入的内容
- 仅在用户明确操作时，向表单字段填充数据

**数据访问：**

- 仅访问表单元素的 DOM 结构，不访问页面的其他内容
- 不读取用户已输入的表单数据
- 不访问页面的其他敏感信息

### 3. storage 权限

**使用理由：**
本扩展需要存储用户的 API 配置信息（科大讯飞 API 密钥）和按钮位置偏好设置，以便：

1. 保存用户配置的 API 密钥，避免每次使用都需要重新输入
2. 保存用户拖拽按钮后的位置，提供更好的用户体验

**具体用途：**

- 存储科大讯飞 API 的 APIPassword（仅存储在浏览器本地）
- 存储用户拖拽按钮后的位置坐标

**数据存储：**

- 所有数据仅存储在浏览器本地（Chrome Extension Storage API）
- 数据不会上传到任何服务器
- 数据不会与第三方共享
- 即使用户清除浏览器数据，这些配置也会被清除

### 4. 远程代码使用说明

**使用理由：**
本扩展需要调用科大讯飞 Spark API（https://spark-api-open.xf-yun.com）来生成表单模拟数据。由于AI模型需要根据表单字段的标签和类型智能生成符合实际场景的模拟数据，因此必须使用远程API服务。

**具体用途：**

- 将表单字段信息（字段名称、类型、标签）发送到科大讯飞 API
- 接收 AI 生成的模拟数据（如姓名、手机号、邮箱、地址等）
- 将生成的数据填充到表单字段中

**数据传输：**

- 仅发送表单字段的元数据（字段名称、类型、标签），不发送用户已输入的数据
- 不发送用户的个人信息
- 不发送页面的其他内容
- API 密钥由用户自行配置，存储在浏览器本地，不会上传到任何服务器

**数据接收：**

- 仅接收 AI 生成的模拟数据（JSON 格式）
- 生成的数据仅用于填充表单，不会存储或传输到其他地方

## 数据使用情况

### 数据收集

本扩展**不收集**任何用户数据，包括：

- 不收集用户的个人信息
- 不收集用户的浏览历史
- 不收集用户输入的表单数据
- 不收集用户的设备信息
- 不收集用户的位置信息

### 数据传输

本扩展仅在以下情况下传输数据：

1. **向科大讯飞 API 发送请求**：

   - 传输内容：表单字段的元数据（字段名称、类型、标签）
   - 传输目的：生成模拟数据
   - 传输方式：HTTPS 加密传输
   - 数据不包含用户个人信息

2. **接收科大讯飞 API 响应**：
   - 接收内容：AI 生成的模拟数据（JSON 格式）
   - 数据用途：仅用于填充表单
   - 数据不存储、不上传、不与第三方共享

### 数据存储

本扩展仅在浏览器本地存储以下数据：

1. **API 配置**：用户配置的科大讯飞 API 密钥（APIPassword）
2. **用户偏好**：按钮位置坐标

所有数据仅存储在浏览器本地，不会上传到任何服务器。

### 数据共享

本扩展**不与任何第三方共享**用户数据。

### 数据安全

- 所有 API 密钥仅存储在浏览器本地
- 数据传输使用 HTTPS 加密
- 不收集、不存储、不上传用户的个人信息
- 用户可随时清除浏览器数据来删除所有配置

## 用户控制

用户完全控制本扩展的行为：

1. **配置控制**：用户可以选择是否配置 API 密钥
2. **使用控制**：用户需要主动点击"自动填充"按钮才会执行填充操作
3. **数据控制**：用户可以随时清除浏览器数据来删除所有配置
4. **权限控制**：用户可以在 Chrome 扩展管理页面随时禁用或卸载本扩展

- ✅ 最小权限原则：仅请求必要的权限
